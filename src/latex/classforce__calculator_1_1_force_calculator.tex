\doxysubsubsection{\texorpdfstring{alpha}{alpha}}
{\footnotesize\ttfamily force\+\_\+calculator.\+Force\+Calculator.\+alpha}



Variable to store the alpha parameter, which is used to scale the direction of the repulsive force. 

If alpha = 0, the repulsive force is computed only considering only the direction of motion. If alpha = 1, the repulsive force is computed only considering the direction of the obstacles. By changing the value of alpha, the repulsive force can be computed considering both the direction of motion and the direction of the obstacles. \mbox{\Hypertarget{classforce__calculator_1_1_force_calculator_a88ce323dffff0f343220b7f9424c7dff}\label{classforce__calculator_1_1_force_calculator_a88ce323dffff0f343220b7f9424c7dff}}
